# Part 1:

## Dependencies:

* [Node](https://nodejs.org) 
* [Gulp](http://gulpjs.com/)
* [Bower](http://bower.io/)

## View Part-1 code:

* git checkout part-1-start
* npm install
* bower install
* gulp serve
* [http://localhost:5000/](http://localhost:5000/)

* bower install
* gulp serve
